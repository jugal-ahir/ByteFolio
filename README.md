## 🚀 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jugal-ahir/ByteFolio.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd ByteFolio
   ```
3. **Install Dependencies**
   ```bash
   yarn
   ```
4. Start your development server
   ```bash
   yarn dev
   ```
